# AI Paperboy

## 1. Project Abstract

### Purpose

* 사용자가 질문으로 요청하면 답변이 있는 뉴스 기사를 스크랩 해주는 서비스

### Function
* 뉴스 스크랩 기능
* 실시간 뉴스 추천 기능
* AI 추천 스크랩 기능
